# Planificador de Procesos (Web Futurista)

App web para simular y visualizar la planificación de procesos tipo sistema operativo.

## Características
- Tabla editable de procesos (PID, llegada, ejecución, prioridad)
- Selección de algoritmo (prioridad, tiempo de ejecución, PID)
- Soporte multiproceso (canales)
- Visualización animada de la ejecución (timeline)
- Tabla de resultados (espera, finalización)
- Exportar resultados a CSV, PDF o copiar tabla
- Estilo futurista (oscuro, acentos neón)

## Instalación y uso

1. Instala dependencias:
   ```bash
   npm install
   ```
2. Corre la app:
   ```bash
   npm start
   ```
3. Abre en tu navegador: [http://localhost:3000](http://localhost:3000)
